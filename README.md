# Power-BI-Sales-Tech-

An interactive Power BI dashboard analyzing sales performance for FY 2026–2027.
Built to track key metrics including Total Sales (16M), Total Cost (12M), Total Profit (4M), and 800+ Transactions.
Includes a bar chart showing quantity sold across 12 product categories, with Monitor (294) being the top seller.
A pie chart breaks down brand-wise sales, led by Intel (19.61%), Samsung (16.34%), and Dell (14.88%).
An interactive map visualizes sum of sales distributed across Indian states.
A supervisor filter panel allows slicing data by 6 sales managers including Roshan Kumar, Ajay Sharma, and others.
Built using Power BI, DAX measures, Excel data source, and Bing Maps integration.
The dashboard helps businesses identify top-performing categories, brands, regions, and supervisors at a glance.
Ideal for sales teams, managers, and analysts looking for quick data-driven insights.
Clone the repo, open the .pbix file in Power BI Desktop, and explore your data interactively! 🚀
